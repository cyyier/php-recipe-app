# レシピ管理ウェブサイト

## 特徴
- **食材量自動調整**: 主要食材の量に基づき、他の食材の量が自動的に調整されます。
- **ステップ別タイマー**: 各調理ステップにタイマーを設定し、ボタンをクリックするだけで計時を開始します。

## 技術スタック
- PHP 7.3
- MySQL 5.7
